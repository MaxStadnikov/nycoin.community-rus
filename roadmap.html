Карта развития

Техническая карта развития сообщества

Наша команда разработчиков состоит из членов сообщества, которые добровольно тратят своё время для выполнения этой дорожной карты. Идеи разделяются открыто среди сообщества, и стратегия определяется на основе консенсуса команды разработчиков.

Планы на 2018:

Первый квартал:

  • Добавление отечественных и зарубежных продавцов которые принимают NewYorkCoin за свои товары и услуги.
  • Выпуск обновлений для кошельков.
    • Обновление кода кошелька newyorkc-qt для удаления ненужных систем оповещения
    • Удаление ошибочных строк кода, на которые непреднамеренно срабатывает антивирусная защита
  • Аудит текущего исходного кода NewYorkCoin
    • Прогресс можно отследить здесь https://github.com/NewYorkCoin-NYC

Второй квартал:

  • Обновление исходного кода NewYorkCoin для введения SegWit и будущей поддержки Lightning Network
  • Выполнение анализа угроз блокчейн NewYorkCoin
    • Реализация патчей для повышения устойчивости к 51% атакам (например, расширенные контрольные точки)
  • Построение библиотеки newyorkcj или сопоставимой bitcoinj библиотеки для достижения следующего:
    • Мультиплатформенные мобильные и декстопные кошельки
    • BitTeller® банкоматы в различных местах приема NYCoin

Третий квартал:

  • Партнёрство с существующими и будущими альткоинами для платежей в интернет-магазинах NewYorkCoin решение задач.
  • Построение плана снижения инфляции NewYorkCoin, чтобы уравновесить необходимость поддержания нулевых или почти нулевых транзакционных сборов, в то же время стимулируя майнеров к добычи блоков NewYorkCoin блокчейн.

Четвёртый квартал:

  • Внедрение плана снижения инфляции после получения консенсуса по сети
  • Исследование атомного обмена сетевые возможности в главной сети

Планы на 2019:

Первый квартал:

  • Внедрение поддержки, интеллектуальных контрактов через Merklized Abstract Syntax Tree («MAST») предложенное сообществом Bitcoin
  • Разработать собственное коммерческое решение для интеграции с NewYorkCoin
